Principais Formatações em HTML

Negrito/Destaque

Nessa frase temos um termo em negrito usando a tag <b>. A tag <b> não é uma tag semântica, apenas de formatação.

Nessa frase temos um termo em destaque usando a tag <strong>. Visualmente não há diferença, mas essa tag possui semântica!

Itálico/Ênfase

Nesta frase, temos um termo em itálico usando a tag <i>. Não é uma tag semântica, apenas de formatação.

Nesta frase temos um termo em ênfase usando a tag <em>. Visualmente não há diferença, mas essa tag possui semântica!

Texto Marcado

Podemos criar também um texto marcado usando a tag <mark>, semelhante a um marca-texto. A tag <mark> possui semântica.

Foi usado style=" background-color: aquamarine;" para dar essa cor ao marcador.

E no outro parágrafo, temos outro texto marcado em cor diferente. Este está na cor padrão.

Texto grande e Pequeno

Estamos criando um texto grande usando a tag <big>, na qual já está obsoleta. E um texto pequeno usando a tag <small>.

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado, usando a tag <del>. A tag <del> é possui semântica!

Texto Inserido

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois. Usamos a tag <ins>. A tag <ins> possui semântica.

Existe também o sublinhado, que não é semântica! A tag <u> é somente para formatação.

Texto Sobrescrito

Para inserir coisas do tipo: x20 + 3. Usamos a tag <sup>.

Texto Subscrito

Para inserir coisas do tipo: h20. Usamos a tag <sub>.

Trechos de Código

Para compartilharmos nossos trechos de código usamos a tag <code>. POssui uma formatação de fácil leitura e uma semântica apropriada!
É interessante usar em conjunto a tag <pre>, que mantém o texto pré-formatado da maneira exata no qual foi escrito, incluindo quebras de linha, espaços e tabulações.

Citações " "

Uma citação é um trecho de um texto escrito ou dito por outra pessoa, que vai ilustrar perfeitamente aquilo que você quer explicar.
Para criar uma citação em HTML usamos a tag <q>. Também podemos criar citações mais longas (em bloco) e que tenham um parágrafo só para si. Nesse caso, colocaremos tudo dentro de <blockquote> </blockquote>. Podemo colocar também um link para o texto original, usando o parâmetro cite="" dentro da tag!

Abreviações

Sempre que quiser escrever uma sigla, mas também quer deixar claro ao usuário (e aos mecanismos de busca) o significado dela, use a tag <abbr>.
O que for escrito no parâmetro title="" da tag <abbr> aparecerá ao colocar o cursos por um determinado tempo em cima da palavra com essa propriedade.

Listas

Há dois tipos de listas: as ordenadas <ol> e as não ordenadas <ul>!

Listas Ordenadas e Seus tipos:

Lista ordenada padrão:

  1. item 1
  2. item 2

Lista ordenada tipo A:

  1. item 1
  2. item 2
  3. Há uma variação do tipo a:
    1. item 1
    2. item 2

Lista ordenada tipo I

  1. item 1
  2. item 2
  3. Há uma variação do tipo i
    1. item 1
    2. item 2

Listas Não Ordenadas e Seus tipos:

Lista não ordenada padrão(disc):

  • item 1
  • item 2

Lista não ordenada circle

  • item 1
  • item 2

  • item 2

Trabalhando com Links

Usamos a tag <a> para criar âncoras de links hyper text

Aqui tem alguns parâmetros utilizados e suas funções:

href=""

HyperText Reference: Colocando a URL completa da página web desejada, cria-se o link propriamente dito.

target=""
Target: Utiliza-se para dizer como link irá se comportar:
  • _self
    • para abrir na mesma página
  • _blank
    • para abrir em outra janela web.
ref=""

Ref: Utiliza-se para "referenciar" o link:

  • external
    • Usado para referenciar links externos.
  • next
    • Usado para referenciar links que levam a próxima página.
  • prev
    • Usado para referenciar links que levam a página anterior.

Links Para Download

Links para download usam a mesma tag <a>, o href="" irá ter o valor do objeto que se deseja fazer o download, o type=""
indicará ao navegador que tipo de arquivo está sendo baixado e o download="" terá valor com o nome do arquivo a ser baixado.
