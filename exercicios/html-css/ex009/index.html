Fazer listar ordenadas usando a tag <ol>(ordened list). É possivel específicar um estilo de lista pelo parâmento type="" e o algarísmo pelo qual a lista vai começar com o parâmetro start="".

Listas Não Ordenadas

  • Comprar pão
  • Comprar Leite
  • Comprar tomate
  • Comprar manteiga
  • Comprar arroz
  • Comprar feijão

Fazer listar não ordenadas usando a tag <ul>(unordened list). É possivel específicar um estilo de lista pelo parâmento type="". Sendo: circle, disk(padrão) ou square.
