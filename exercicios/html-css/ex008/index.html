Principais Formatações

Negirto/Destaque

Nessa frase temos um termo em negrito usando a tag <b>. A tag <b> não é uma tag semântica, mas sim de formatação!

Nessa frase temos um termo em destaque usando a tag <strong>! Visualmente não há diferença, mas semânticamente, a tag <strong> da a formatação e o significado que queremos pro código!!

Itálico/Ênfase

Nesta frase, temos um termo em itálico usando a tag <i>. Não é uma tag semântica!

Nesta frase temos um termo em ênfase usando a tag <em>. Esse traz uma formatação e significado(semântica)!

Texto Marcado

Podemos criar também um texto marcado usando a tag <mark>. Semelhante a um marca-texto.

E no outro parágrafo, temos outro texto marcado no final.

Texto Grande e Pequeno

Estamos criando um texto grande usando a tag <big>, na qual já está obsoleta. E um texto pequeno usando a tag <small>, nesse parágrafo.

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele dever ser lido, mas não considerado. Usando a tag <del>. Tag semântica!

Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. Usando a tag <ins>! Tag semântica!

Existe também o sublinhado que é com a tag <u>, que é não semântica!Tag somente para formatação.

Texto Sobrescrito

Para inserir coisas do tipo x20 + 3. Usando a tag <sup>.

Texto Subscrito

Para inserir coisas do tipo H20. Usando a tag <sub>.
