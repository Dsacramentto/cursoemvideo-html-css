Fazer listar ordenadas usando a tag <ol>(ordened list). É possivel específicar um estilo de lista pelo parâmento type="" e o algarísmo pelo qual a lista vai começar com o parâmetro start="".

Fazer listar não ordenadas usando a tag <ul>(unordened list). É possivel específicar um estilo de lista pelo parâmento type="". Sendo: circle, disc(padrão) ou square.
